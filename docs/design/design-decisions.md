# Design Decisions
This file explains WHY design choices were made, not just what they
are.
Every major decision is traceable to either a field research insight or
a
technical constraint.
## Decision Log
| Decision | Reason | Source |
|----------|--------|--------|
| Mobile-first layout | School staff access via phones, not desktops |
Field research: all schools visited had teachers using smartphones |
| Minimal text input on mobile forms | Low-literacy accommodation |
Headmaster interview: 'Many staff find typing on phones difficult' |
| Category dropdown instead of free text | Enables algorithmic
classification | Architecture requirement |
| 2-hour edit window (not 1 hour) | Shared device usage in schools |
Questionnaire finding: teachers may not return to device within 1 hour
|
| Telugu labels for categories | Primary language of users | Field
research: English labels caused confusion in School 2 walkthrough |
See [../field-research/insights/design-changes-from-field.md](../fieldresearch/insights/design-changes-from-field.md) for full list.
