# SETU — Smart Escalation & Tracking Utility
> Grievance tracking system for TGTWREIS Gurukulam tribal welfare
schools, Telangana.
> Built for FOSS Hack 2026.
## The Problem
TGTWREIS Gurukulam schools house thousands of tribal students in
residential schools
across Telangana. When infrastructure fails — water, electricity,
safety — school staff
report the issue on paper. There is no tracking, no priority system,
and no accountability.
Issues routinely go unresolved for weeks or months.
## What SETU Does
SETU is a full-stack web application that gives school staff, district
officials, and
state administrators a shared, transparent system for reporting and
resolving issues.
- School staff submit issues with photos and descriptions
- AI keyword algorithm classifies urgency automatically
- Priority score escalates automatically if no action is taken
- Officials have dashboards with real-time accountability metrics
- Every action is immutably logged — nothing can be silently deleted
## Stack
Next.js · Node.js/Express · PostgreSQL · No proprietary APIs
